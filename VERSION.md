## v3.0 (2026-02-27) - 当前稳定版

### 架构重构
- 分离核心逻辑与执行层
- `core/trader.py` - 核心交易逻辑
- `core/executor.py` - 执行层接口
  - `BacktestExecutor` - 回测执行器
  - `LiveExecutor` - 实盘执行器

### 新增功能
- 资金划转支持（统一账户模式）
- 修复API签名问题

### 核心特性
- 趋势判断: MA20斜率
- 技术指标: RSI/MACD/布林带/ATR/ADX
- 信号过滤: 只在趋势明显的市场中交易
- 移动止损 + 分批止盈
- 加仓策略: 同向信号加仓，反向信号平仓
- 每月提取收益
- 爆仓保护

### 运行命令

```bash
# 回测
python main.py --backtest --days 365

# 实盘
python main.py --live

# 单次分析
python main.py --once
```

## 资金管理规则

### 提取+补充机制
- 每月检查余额：
  - **余额>100U**：提取超出部分
  - **余额<100U**：从已提取资金补充回100U
- 仓位计算始终基于100U
- 最终收益 = 余额100U + 累计提取 - 初始100U
