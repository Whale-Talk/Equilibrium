## v3.0.1 更新 - Bug修复 (2026-02-27)

### 架构重构
- 分离核心逻辑与执行层
- `core/trader.py` - 核心交易逻辑
- `core/executor.py` - 执行层接口
  - `BacktestExecutor` - 回测执行器
  - `LiveExecutor` - 实盘执行器

### 优势
- 回测与实盘共用同一套核心逻辑
- 保证100%一致性
- 代码更模块化，易于维护

### 运行命令

```bash
# 回测
python main.py --backtest --days 365

# 实盘
python main.py --live

# 单次分析
python main.py --once
```

## v2.4 更新 - 爆仓保护

### 新增功能
- **爆仓价格计算**：开仓时计算爆仓价并监控
- **加仓时更新爆仓价**：根据新均价重新计算
- **爆仓强制平仓**：价格触及爆仓价时触发

### 爆仓公式
- 多头：`爆仓价 = 开仓价 × (1 - 1/杠杆 + 0.5%)`
- 空头：`爆仓价 = 开仓价 × (1 + 1/杠杆 - 0.5%)`

### 安全机制
- 止损始终在爆仓价相反方向
- **保证先触及止损，不会爆仓**

## 资金管理规则

### 提取+补充机制
- 每月检查余额：
  - **余额>100U**：提取超出部分
  - **余额<100U**：从已提取资金补充回100U
- 仓位计算始终基于100U
- 最终收益 = 余额100U + 累计提取 - 初始100U
