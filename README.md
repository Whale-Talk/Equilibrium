# TradingAgents BTC 量化交易系统 - BTC3.0 版本

## 版本说明

BTC3.0 是基于TradingAgents的智能量化交易系统，采用AI驱动的动态止损止盈策略。

## 核心特性

- **多智能体架构**: 技术分析师 + 新闻分析师 + 多头研究员 + 空头研究员 + 交易员 + 风控经理
- **AI动态止损止盈**: 由AI根据支撑/阻力位决定具体价格
- **智能风控**: 风控经理审核并可调整止损止盈
- **技术指标**: RSI/MACD/布林带/ATR/ADX
- **Telegram通知**: 实时推送交易信号

## AI决策流程

```
1. 技术分析师 → 技术分析报告 (RSI/MACD/布林带/ADX)
2. 新闻分析师 → 新闻分析报告
3. 多头研究员 → 提出做多理由
4. 空头研究员 → 提出做空理由
5. 交易员 → 决定 action/leverage/止损价格/止盈价格
6. 风控经理 → 审核并调整止损止盈
```

## 止损止盈策略

- **AI决定**: 由交易员根据技术分析给出具体价格
- **风控调整**: 风控经理审核并可调整
- **默认值**: 如果AI未给出，使用布林带+ATR计算
  - 止损 = 布林下轨 - 1.5倍ATR
  - 止盈 = 2.5倍盈亏比

## 运行命令

```bash
# 模拟盘运行
python main.py

# 运行一次分析
python main.py --once

# 回测 (AI驱动，较慢)
python main.py --backtest --days 7
```
