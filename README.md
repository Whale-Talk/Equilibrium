# BTC 量化交易系统 - BTC3.0 版本

## 版本说明

BTC3.0 是基于技术指标的量化交易系统，包含两个版本：

- **v1**: 基础策略（不加仓）
- **v2**: 加仓策略（同向加仓 + 反向平仓 + 每月提取收益）

## 核心特性

- **趋势判断**: 基于MA20斜率判断市场趋势
- **技术指标**: RSI/MACD/布林带/ATR/ADX
- **信号过滤**: 只在趋势明显的市场中交易
- **移动止损**: 止盈后止损移至保本
- **分批止盈**: TP1=1.5倍, TP2=3倍
- **加仓策略** (v2): 同向信号加仓，反向信号平仓
- **每月提取收益** (v2): 余额超过100U时提取
- **Telegram通知**: 实时推送交易信号

## 策略说明

### v1: 基础策略

- 同向信号: 忽略
- 反向信号: 忽略
- 只在趋势明确时交易

### v2: 加仓策略（推荐）

- 同向信号 → 加仓
- 反向信号 → 平仓
- 每月提取收益（保持100U基础）

## 运行命令

```bash
# 模拟盘运行
python main.py

# 运行一次分析
python main.py --once

# 回测
python main.py --backtest --days 365
```
